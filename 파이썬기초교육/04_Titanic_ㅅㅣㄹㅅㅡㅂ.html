Markdown 알아보기¶

마크다운은 웹에서 텍스트의 스타일을 지정하는 방법으로 문서의 표시를 제어한다. 굵게 또는 기울임 꼴로 단어 서식을 지정하고, 이미지를 추가하고, 목록을 만드는 작업은 Markdown에서 제공한다. 대부분 Markdown은 # 또는 * 와 같이 일부 알파벳이 아닌 문자가 포함 된 일반 텍스트이다.

Features 변수명 설명¶

  • survival: Survival (0 = No; 1 = Yes)
  • pclass: Passenger Class (1 = 1st; 2 = 2nd; 3 = 3rd)
  • name: Name
  • sex: Sex
  • age: Age
  • sibsp: Number of Siblings/Spouses Aboard
  • parch: Number of Parents/Children Aboard
  • ticket: Ticket Number
  • fare: Passenger Fare
  • cabin: Cabin
  • embarked: Port of Embarkation (C = Cherbourg; Q = Queenstown; S = Southampton)
